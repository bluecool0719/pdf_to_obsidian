# PDF → Obsidian 자동 동기화 스크립트

PC의 과목별 폴더에 있는 PDF 파일을 Obsidian 볼트의 해당 과목 폴더로 자동으로 복사해주는 스크립트입니다.

---

## 기능

- PC 전체를 탐색해 볼트 과목 폴더명과 일치하는 폴더를 자동으로 찾아 복사
- 학기 폴더 구조 (`2-1/기초생화학`, `2-2/기초생화학` 등)에 관계없이 동작
- 복사된 PDF는 볼트 내 `(과목명) 강의록/` 폴더에 저장 (최초 실행 시 폴더 자동 생성)
- 이미 복사된 파일은 건너뜀 (중복 방지)
- 매칭 후보가 여러 개일 경우 직접 선택 가능
- 한 번 매칭된 폴더 쌍은 `pdf_obsidian_matches.json`에 저장되어 다음 실행 시 재사용

---

## 요구사항

- Python 3.9 이상
- 외부 라이브러리 설치 불필요 (표준 라이브러리만 사용)

---

## 설정

스크립트 상단의 세 가지 항목만 본인 환경에 맞게 수정하세요.

```python
# PDF를 찾을 탐색 범위 (여러 개 지정 가능)
SEARCH_ROOTS = [
    r"C:\Users\사용자이름\Documents",
    r"C:\Users\사용자이름\Desktop",
]

# Obsidian 볼트 경로
VAULT_ROOT = r"C:\Users\사용자이름\볼트경로"

# 탐색 최대 깊이 (2-1/과목명 구조라면 2면 충분)
MAX_DEPTH = 4
```

### 볼트 경로 확인 방법

1. Obsidian 실행
2. 좌측 하단 볼트 이름 클릭 → **Open vault folder**
3. 탐색기 주소창에서 경로 복사

> 경로 앞의 `r`은 반드시 유지하세요. Windows 경로의 백슬래시(`\`) 문제를 방지합니다.

---

## 사용법

```bash
python pdf_to_obsidian.py
```

또는 스크립트 파일을 더블클릭해서 실행합니다.

```
│   └── 기초약리학 강의록/

        └── 세포구조_족보.pdf
```

## 매칭 캐시 초기화 방법

학기가 바뀌어 폴더 경로가 달라졌을 경우, 스크립트 옆에 생성된 `pdf_obsidian_matches.json`을 메모장으로 열어 해당 과목 줄만 삭제하면 다음 실행 시 재탐색합니다.

```json
{
  "기초생화학": ["C:\\Users\\destr\\Documents\\2-1\\기초생화학"],
  "기초약리학": ["C:\\Users\\destr\\Documents\\2-1\\기초약리학"]
}
```
